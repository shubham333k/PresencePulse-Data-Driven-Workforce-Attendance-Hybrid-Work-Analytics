📊 PresencePulse: Workforce Attendance & Hybrid-Work Analytics




🏛️ Project Overview

PresencePulse is a high-impact Power BI dashboard designed to provide HR leaders and management with a 360-degree view of employee attendance, remote work (WFH) trends, and sick leave patterns. By transforming raw attendance logs into interactive visual intelligence, this project enables organizations to monitor workforce engagement, optimize hybrid-work policies, and identify absenteeism hotspots.

High-Impact Findings

•
Regime Analysis: Identified specific months with 20% higher absenteeism, correlating with seasonal health trends.

•
Hybrid-Work Optimization: Discovered that WFH productivity peaks on specific days, enabling better scheduling of in-office meetings.

•
Engagement Tracking: Pinpointed departments with inconsistent attendance patterns, allowing for proactive HR intervention.




🛠️ Technical Architecture

Data Workflow

1. Data Ingestion: Processed raw attendance datasets from Microsoft Excel.

2. ETL & Transformation: Utilized Power Query for advanced data cleaning and schema modeling.

3. Calculations: Developed custom DAX measures for KPIs like Presence %, WFH %, and Sick Leave %.

4. Visualization: Built an interactive multi-page report with slicers for date ranges, departments, and employee-specific views.

Tech Stack

• BI Tool: Power BI Desktop

• Data Cleaning: Power Query (M Language)

• Analytics: DAX (Data Analysis Expressions)

• Source: Microsoft Excel / CSV


💡 Business Use Cases

• HR Workforce Monitoring: Real-time tracking of employee presence across the organization.

• Remote Work Policy Design: Data-backed decisions on "Work from Home" vs "Work from Office" ratios.

• Resource Planning: Identifying peak absenteeism periods to ensure adequate staffing.
